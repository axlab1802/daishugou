# UIデザイン仕様

## 1. 全体トーン
- 中世の酒場をイメージした配色とカード演出
- 高コントラストでスマホでも視認性を確保

## 2. 画面構成
- ヘッダー: タイトル表示
- 情報バー: 対戦人数・革命状態
- モード切替: ローカル / オンライン
- オンラインロビー: 名前、ルームコード、最大人数
- テーブル: 最大6人の座席配置
- 操作バー: 出す / パス

## 3. 主要UI要素
- カード: 酒アイコンと名称、ランク、背景グラデーション
- 角ラベル: 上下の数字横に酒名を同サイズで表示（モバイル可読性優先）
- コンボ演出: 画面中央に強調テキスト
- 革命インジケータ: 状態を色で判別

## 4. モバイル対応
- 720px以下で縦並び
- 座席を縦方向に並べ、手札を画面下に固定
- タップ領域は44px以上を確保

## 5. ニックネームの自動入力
- ロビーの名前欄に保存済みニックネームを自動反映
- 作成/参加時に新しい名前を保存

## 6. 6人レイアウト
- `bottom`, `left`, `right`, `top`, `top-left`, `top-right` の6席
- オンライン時は自分を `bottom` に固定
