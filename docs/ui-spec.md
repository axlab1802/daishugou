# UIデザイン仕様

## 1. 全体トーン
- **中世の酒場** × **トランプ** の世界観
- 背景は深いブラウン系グラデーション
- タイポグラフィはセリフ系（Garamond系）でクラシック感

## 2. レイアウト構成
1. ヘッダー（タイトル）
2. 情報バー（枚数・革命状態）
3. モード切替（ローカル / オンライン）
4. CPU人数セレクタ + ゲーム開始ボタン
5. 追加ルール設定（チェックボックス）
6. オンラインロビー（名前/ルームコード/作成/参加/開始）
7. テーブルレイアウト（最大6席）
8. 場（出されたカード）
9. メッセージ（ターン通知）
10. 自分の手札
11. アクションボタン（出す / パス）

## 3. カードデザイン

### 3.1 共通仕様
- 角丸: 12px
- 枠線: 2px solid #2c1b10
- 影: 0 8px 16px rgba(0,0,0,0.2)
- **トランプのイメージ**: 左上/右下にランク表示
- 自分の手札は左上からランク昇順で並ぶ

### 3.2 レイアウト一括変更
- `tools/card_layout.json` を編集するとカードSVGの配置を一括変更できる
- 生成コマンド: `tools/generate_cards.py`
- ゲームでは `assets/generated` のSVGを表示する
- ジョーカーのみ `assets/joker.png` を直接表示する

### 3.3 中央アート
- 中央フレーム内に **「中世風の油絵」** の文字を表示
- その下に酒名（日本語、例: ビール）を表示
- フレームは木枠風グラデーション

### 3.4 ランク別背景色
- docs/specification.md のグラデーション配色を適用

### 3.5 状態
- ホバー: Y軸 -6px
- 選択: Y軸 -12px + 発光グロー

## 4. カード裏面
- 斜めストライプ + ダークグラデーション
- 中央にイニシャル「Y」

## 5. ボタン
- Primary: 金色系グラデーション（出す！）
- Ghost: 半透明白枠（パス）

## 6. 派手演出（GSAP）
- コンボ発動・革命・場流し時に中央オーバーレイ表示
- メッセージ更新時にフェード & スライド

## 7. 最大6人レイアウト
- 座席配置: bottom(自分) / left / top-left / top / top-right / right
- CPU人数に応じて座席を表示/非表示
- 座席には名前・残り枚数・裏向きカードを表示

## 7.1 オンラインロビー
- ルーム作成、参加、開始、退出の操作を提供
- 参加者リストとオーナー表示

## 7.2 追加ルール設定
- 追加ルールのON/OFFをチェックボックスで切り替え
- 未実装ルールは無効表示（編集不可）

## 8. レスポンシブ
- カード幅・高さは `clamp()` で可変
- 720px以下でボタン縦並び

## 9. ゲーム中の簡易表示
- ゲーム開始後は設定UI（ヘッダー/情報バー/モード/設定/ロビー）を非表示
- 場・相手・自分の手札と操作のみを表示

## 10. 効果音
- 操作・演出に効果音を割り当てる
- `assets/sfx/sfx.json` で音の割り当てを管理
