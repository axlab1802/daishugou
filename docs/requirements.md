# 酔いどれ大富豪 要件定義書

## 1. プロジェクト概要

### 1.1 プロジェクト名
**酔いどれ大富豪（Yoidore Daifugo）**

### 1.2 コンセプト
トランプの大富豪をベースに、カードを「お酒」に置き換えたカードゲーム。
酒の強さ（アルコール度数イメージ）でカードの強弱が決まり、特殊コンボで逆転が可能。

### 1.3 ターゲットユーザー
- カジュアルゲーマー
- 大富豪ルールを知っている人
- 居酒屋・お酒が好きな人
- 1人でサクッと遊びたい人

### 1.4 プラットフォーム
- Webブラウザ（PC/スマートフォン対応）
- HTML5 + CSS3 + JavaScript（バニラJS）

## 2. 機能要件

### 2.1 ゲームモード

| ID | 機能 | 優先度 | 説明 |
|----|------|--------|------|
| F-001 | CPU対戦モード | 必須 | 1人 vs CPU（kiyoshi）の対戦 |
| F-002 | 難易度選択 | 任意 | CPU の強さを選択可能 |
| F-003 | オンライン対戦 | 必須 | 2〜6人でのオンライン対戦 |
| F-004 | CPU人数選択 | 必須 | 1人+CPU1〜5人で対戦可能 |

### 2.2 ゲームフロー

| ID | 機能 | 優先度 | 説明 |
|----|------|--------|------|
| F-010 | タイトル画面 | 必須 | ゲーム開始、ルール確認ボタン |
| F-011 | ルール説明画面 | 必須 | カードの強さ、コンボ一覧を表示 |
| F-012 | ゲーム画面 | 必須 | メインのプレイ画面 |
| F-013 | 結果画面 | 必須 | 勝敗結果とリプレイボタン |

### 2.3 ゲームプレイ機能

| ID | 機能 | 優先度 | 説明 |
|----|------|--------|------|
| F-020 | カード配布 | 必須 | 各プレイヤーに10枚配布 |
| F-021 | 先攻決定 | 必須 | 水（3）を持っている方が先攻 |
| F-022 | カード選択 | 必須 | 手札から出すカードを選択 |
| F-023 | カード出し | 必須 | 選択したカードを場に出す |
| F-024 | パス機能 | 必須 | カードを出さずにパス |
| F-025 | 場流し | 必須 | 全員パスで場をリセット |
| F-026 | 勝敗判定 | 必須 | 先に手札を出し切った方が勝ち |
| F-027 | 追加ルール設定 | 必須 | 追加ルールをON/OFFで切り替え可能 |

### 2.4 特殊ルール機能

| ID | 機能 | 優先度 | 説明 |
|----|------|--------|------|
| F-030 | 複数枚出し | 必須 | 同じランクのカードを複数枚同時に出せる |
| F-031 | 革命 | 必須 | 同じカード4枚出しで強さ逆転 |
| F-032 | 特殊コンボ | 必須 | 特定の組み合わせで通常より強いカードに勝てる |
| F-033 | コンボヒント | 必須 | 使用可能なコンボがある時、該当カードを光らせる |
| F-034 | ジョーカーカード | 必須 | ジョーカーは1枚のみ、2より強い最上位カードとして扱う |

### 2.5 UI/UX機能

| ID | 機能 | 優先度 | 説明 |
|----|------|--------|------|
| F-040 | カードアニメーション | 必須 | カード選択・出し時のアニメーション |
| F-041 | コンボ演出 | 必須 | コンボ発動時の派手な演出 |
| F-042 | ターン表示 | 必須 | 現在のターンを明示 |
| F-043 | 残り枚数表示 | 必須 | 各プレイヤーの残り手札枚数 |
| F-044 | 革命インジケーター | 必須 | 革命中であることを明示 |
| F-045 | レスポンシブ対応 | 必須 | スマートフォンでも快適にプレイ可能 |
| F-046 | カードアート表記 | 必須 | カード中央に「中世風の油絵」と酒名の文字を表示 |
| F-047 | 最大6人レイアウト | 必須 | 6人対戦時の座席レイアウトを表示 |
| F-048 | オンラインロビー | 必須 | ルーム作成/参加/開始ができるロビーUI |
| F-049 | カードレイアウト一括変更 | 必須 | レイアウト定義から一括でカードSVGを生成できる |
| F-050 | ゲーム中の簡易表示 | 必須 | ゲーム開始後は設定UIを非表示にして場/手札/操作のみ表示 |
| F-051 | 効果音 | 必須 | ゲームの操作/演出に効果音を再生する |

## 3. 非機能要件

### 3.1 パフォーマンス

| ID | 要件 | 目標値 |
|----|------|--------|
| NF-001 | 初回ロード時間 | 3秒以内 |
| NF-002 | カード操作レスポンス | 100ms以内 |
| NF-003 | アニメーション | 60fps維持 |

### 3.2 対応環境

| ID | 要件 | 詳細 |
|----|------|------|
| NF-010 | ブラウザ | Chrome, Safari, Firefox, Edge（最新版） |
| NF-011 | デバイス | PC, タブレット, スマートフォン |
| NF-012 | 画面サイズ | 320px〜1920px |

### 3.3 アクセシビリティ

| ID | 要件 | 詳細 |
|----|------|------|
| NF-020 | カラーコントラスト | WCAG AA準拠 |
| NF-021 | タッチ操作 | タップ領域44px以上確保 |

---

## 4. 制約条件

- 外部ライブラリは最小限に抑える（バニラJS推奨）
- サーバーサイド不要（完全クライアントサイド）
- オフラインでも動作可能
